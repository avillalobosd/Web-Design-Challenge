Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplis this analysis, we first pulled data from the OpenWeatherMap API to assegle a dataset on over 500 cities

After assembling the dataset, we used MatPlotLib to plot various aspects of the weather vs. latitud. Factors we looked at included: temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations creates as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed
